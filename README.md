# binge-buddy-api

### dotnet core REST api
API back-end for Binge Buddy web app. Project https://github.com/yegengovender/binge-buddy

## Project setup
```
dotnet install
```

### Compiles and hot-reloads for development
```
dotnet watch
```

### Compiles and minifies for production
```
dotnet build
```

### Start the API
```
dotnet run
```
